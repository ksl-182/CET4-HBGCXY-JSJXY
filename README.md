📚 项目简介
本项目通过 Python 脚本自动生成了 10 篇不同话题的四级作文，将 1500 + 核心词汇自然地嵌入到完整的句子和段落中。每篇作文都采用了四级考试中最常用的句型和结构，并使用颜色高亮区分不同类型的语言元素，让词汇学习更加高效和有趣。
✨ 主要特点
真实语境学习：词汇不是孤立的列表，而是融入完整的作文中
高频词汇覆盖：精选 1500 + 四级考试核心词汇
话题全面：涵盖四级作文最常考的 10 个话题
双语版本：提供中文版（带释义）和英文版（纯英文）
颜色高亮：四种颜色区分核心词汇、关键句型、过渡词和高频短语
词汇索引：每篇作文后附完整词汇索引，方便查阅和复习

🚀 使用方法
下载文件：点击上方 "Code" 按钮，选择 "Download ZIP" 下载整个项目
选择版本：根据你的学习需求选择中文版或英文版
学习方法：
先通读整篇作文，理解整体意思
重点关注高亮的核心词汇和短语
背诵关键句型，尝试用它们造句
利用词汇索引进行复习和自测
模仿作文结构，练习写同话题的作文
🛠️ 技术实现
数据来源
从 GitHub 两个开源仓库获取四级词汇：
yiyisheh/CET4-vocabulary：6825 个高频词（CSV 格式）
vxiaozhi/vocabulary-book-by-deepseek：3320 个完整词表（JSON 格式，含释义）
合并去重后精选 1500 个最核心的词汇
生成脚本
使用 Python 编写generate_cet4_essays.py脚本
将 1500 个词汇随机分配到 10 个话题，每篇约 150 个词汇
自动生成语法正确、逻辑连贯的作文
实现四种颜色的高亮效果
自动生成每篇作文的词汇索引表
格式转换
使用 Pandoc 将 Markdown 格式转换为 Word 文档 (.docx)
保留所有格式和颜色高亮
