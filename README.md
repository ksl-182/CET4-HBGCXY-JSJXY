# CET-4 四级核心词汇作文集

一个自动生成的四级核心词汇作文集，覆盖750高频词汇，帮助考生在真实语境中学习和记忆单词。

## 📚 项目简介

本项目通过Python脚本自动生成了10篇不同话题的四级作文，将1500+核心词汇自然地嵌入到完整的句子和段落中。每篇作文都采用了四级考试中最常用的句型和结构，并使用颜色高亮区分不同类型的语言元素，让词汇学习更加高效和有趣。

## ✨ 主要特点

- **真实语境学习**：词汇不是孤立的列表，而是融入完整的作文中
- **高频词汇覆盖**：精选750四级考试核心词汇
- **话题全面**：涵盖四级作文最常考的10个话题
- **双语版本**：提供中文版（带释义）和英文版（纯英文）
- **颜色高亮**：四种颜色区分核心词汇、关键句型、过渡词和高频短语
- **词汇索引**：每篇作文后附完整词汇索引，方便查阅和复习

## 🚀 使用方法

1. **下载文件**：点击上方"Code"按钮，选择"Download ZIP"下载整个项目
2. **选择版本**：根据你的学习需求选择中文版或英文版
3. **学习方法**：
   - 先通读整篇作文，理解整体意思
   - 重点关注高亮的核心词汇和短语
   - 背诵关键句型，尝试用它们造句
   - 利用词汇索引进行复习和自测
   - 模仿作文结构，练习写同话题的作文

## 🛠️ 技术实现

### 数据来源

- 从GitHub两个开源仓库获取四级词汇：
  - [yiyisheh/CET4-vocabulary](https://github.com/yiyisheh/CET4-vocabulary)：6825个高频词（CSV格式）
  - [vxiaozhi/vocabulary-book-by-deepseek](https://github.com/vxiaozhi/vocabulary-book-by-deepseek)：3320个完整词表（JSON格式，含释义）
- 合并去重后精选1500个最核心的词汇

### 生成脚本

- 使用Python编写`generate_cet4_essays.py`脚本
- 将1500个词汇随机分配到10个话题，每篇约150个词汇
- 自动生成语法正确、逻辑连贯的作文
- 实现四种颜色的高亮效果
- 自动生成每篇作文的词汇索引表

### 格式转换

- 使用Pandoc将Markdown格式转换为Word文档(.docx)
- 保留所有格式和颜色高亮
